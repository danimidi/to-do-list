# to-do-list
Web application implemented with Vue. It consists of a CRUD that allows you to add, edit, delete and mark as complete the items of a task list.
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
